Post 1: Introducción a los Grafos

Un grafo es una estructura de datos utilizada para representar relaciones entre entidades. Un grafo está compuesto por:

  • Vértices o nodos (V): representan entidades u objetos.
  • Aristas o edges (E): representan conexiones o relaciones entre nodos.

Tipos de Grafos

  • Grafo no dirigido: las aristas no tienen dirección.
  • Grafo dirigido (dígrafo): las aristas tienen dirección y se llaman arcos.
  • Grafo ponderado: cada arista tiene un valor (peso o costo).
